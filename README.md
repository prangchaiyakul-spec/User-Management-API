# User Management API Automation Testing

โปรเจกต์ทดสอบอัตโนมัติ (Automated Testing) ฝั่ง API หลังบ้านด้วย Cypress โดยใช้แนวคิด Data-Driven Testing (DDT) ร่วมกับข้อมูล JSON (Fixtures) เพื่อทดสอบลอจิกและขอบเขตความปลอดภัยระบบ

## Tech Stack

- **Automation Framework:** Cypress
- **Language:** JavaScript
- **Test Data Format:** JSON (Fixtures)

---

## Installation and Execution

1. Install project dependencies:

```bash
npm install

1.Open Cypress Test Runner:
npx cypress open

2.Run tests in headless mode:
npx cypress run
```
